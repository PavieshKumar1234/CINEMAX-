# CINEMAX - Movie Intelligence Dashboard

CINEMAX is a Python Streamlit web application used to analyze movie ratings and display movie information in an interactive dashboard. It helps users view movie details, ratings, genres, and visual insights in a simple and attractive way.

## Project Overview

This project is developed using Python and Streamlit. It uses Pandas for data handling, Requests for fetching movie information, and Plotly for creating interactive visualizations.

## Features

* Movie rating analysis dashboard
* Interactive charts and graphs
* Movie details display
* Genre-wise analysis
* User-friendly Streamlit interface
* Clean and simple design

## Technologies Used

* Python
* Streamlit
* Pandas
* Requests
* Plotly

## Requirements

Install the required packages using:

```bash
pip install -r requirements.txt
```

## Run the Project

To run the project, use this command:

```bash
streamlit run movie.py
```

## Files in this Repository

* `movie.py` - Main Streamlit application file
* `requirements.txt` - Required Python packages
* `README.md` - Project explanation file

## Purpose of the Project

The main purpose of this project is to analyze movie ratings and present the data in a visual and interactive format. It is useful for understanding movie performance, rating trends, and genre-based insights.
